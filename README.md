# Static Page Saver

一个用于把 ChatGPT 对话页面保存为静态网页的 Edge 扩展，目标是尽可能复刻原页面样式。

## 使用方法（最简）
1. 打开 Edge，进入 `edge://extensions/`。
2. 开启右上角「开发者模式」。
3. 点击「加载解压缩的扩展」。
4. 选择本项目源码所在的固定目录（建议不要随意移动目录位置）。
5. 导入后即可使用。
